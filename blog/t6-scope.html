Scope

Scope and Javacript

In JavaScript scope can be defined as having two categories, 'Local' and 'Global.' Local scope typically defines a variable that is defined within a function, as such it is exclusive to the function it is nested within. Global scope is when a variable is defined outside of a function, this allows it to be used in any further function or variable. One exception to these two definitions is called 'Automatically Global', this is when a variabel is called earlier than it is defined within a function.

for example:

var animal = 'penguin'; // global scope
function findAnimal() { var animal = "ocelot"; // local scope }
