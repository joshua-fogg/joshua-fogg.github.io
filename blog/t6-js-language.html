In the JavaScript language, describe all the use cases for each of:

parentheses ()

Parentheses have a few uses in scripting. Probably the most familiar way of using them is for math equations. Typically we used them in this manner to give structure to logical arguments by compartmentalising information and enabling information to be nested within other arguments.
Parentheses can also be used to invoke a switch statement or function by describing parameters to be passed to a logic statement. Parenthesis are also used to describ function and object data keys or when structuring logical arguments and parameters. Examples of these include:

function()

function(p1, ..., pN)

Brackets []

Brackets are typically used within Javascript to contain array data sets and allow access to array data through array indexes. Array indexes describe data value(s) placment within the array. Examples include:

var answers = ['yes','no','maybe']

var Bed = [[1,2],[3,4],['bed','camel']]

Bed[2][1] = 'camel'

single quotes '' double quotes ""

Quotes both single and double are used in JavaScript to define strings, which are stores of text. Because strings must be written within quotes, JavaScript will misunderstand strings with quotes within it if the same quote type is used. In teh example below the repeated use of double quotes leads the in text quote of "Vikings" to be read as a variable:

var y = "We are the so-called "Vikings" from the north."

var y = "We are the so-called 'Vikings' from the north."

var y = 'We are the so-called "Vikings" from the north.'

Back-Ticks ``

Back-Ticks have been introduced with ES6, a trademarked scripting-language specification standard by ECMA International. Back-Ticks allow both single and double quotes to utilised in a string as well as allowing multi line strings to be converted into a single line by triggering a template literal.
