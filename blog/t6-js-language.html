Javascript

In the JavaScript language, describe all the use cases for each of:

parentheses ()

Parentheses have a few uses in scripting. Probably the most familiar way of using them is for math equations. Typically we used them in this manner to give structure to logical arguments by compartmentalising information and enabling information to be nested within other arguments.
Parentheses can also be used to invoke a switch statement or function by describing parameters to be passed to a logic statement. Parenthesis are also used to describ function and object data keys or when structuring logical arguments and parameters. Examples of these include: function() function(p1, ..., pN) structure(expression)

Brackets []

Brackets are typically used within Javascript to contain array data sets and allow access to array data through array indexes. Array indexes describe data value(s) placment within the array. Examples include: var answers = ['yes','no','maybe'] var Bed = [[1,2],[3,4],['bed','camel']] Bed[2][1] = 'camel'
