HTML, CSS and DOM

  • Grid Based Design:


    • Grid based design is a design process, where a web pages is broken down into boxes that stack into rows and columns. These boxes can then be filled with page content. The grid design process allows for a clarity in designing and creating a 2-D website. Typically a 12-column grid is used as it allows the page to be broken into 2, 3, 4 and 6 sections easily.

  • Responsive Webpages, what's the deal:


    • Responsive web pages are designed to maintain a consistant user experience across multiple resolutions. This means a website with responsive webpages will adjust to the users screen size helping the user to have continued ease of use across multiple resolutions.
