Control Flow and Loops:

Control Flow is an important concept in computer science, it is the order in which individual statements, instructions or functions are looked at and executed.

Imagine that before you went to bed each night, you wrote a list of things to do the next day after you wake up. This list included activities like brushing your teeth, getting dressed, having breakfast and heading to work.

Now if you were to write these in a radom order you may end up dressing once you have left the house. However if you were to prioritise these activities then you could ensure you don't rock up to work without first being ready for the day.

This prioritisation is '' Control Flow", it causes priorities and order for computers that are working through a script of instructions.

Loops are a type of code that will repeat until interrrupted or told to stop. Loops are great for reducing the amount of coding required to do the same steps. Loops give an efficient way to reduce the amount of code needed to repeat certain steps.

For instance in the previous example a coding loop would be the equivalent of writing out a list once and knowing to re-follow the same list each day, possibly with a few adjustments over teh day to day. So rather than writing a new list each night you could write out a list once and use it each day.


Data Accessing from Arrays and Objects:

Javascript utilises "Objects" and "Arrays", both of which can hold different forms of data. Objects are a variable that can hold defined data lists, for example an object variable called car can hold data for the car type, model and colour:

var car = {type:"Fiat", model:"500", color:"white"};

Accessing object data can be done by calling the object's properties or Key-values. Using the above example you could access it's data with the following:

car.model or car["model"]

Arrays can be used to store multiple values in a single variable. Arrays can be nestled also, allowing multiple bits of data to be stored relative to a value.

var car = ["Fiat", 500, "White"];

Array data can be accessed by using the zero based numeric index, which value starts at [0] for the data entry.

cars[0]; which would return "Fiat"

Additionally, object data can be accessed using dot notation and bracket notation. Arrays rely solely on bracket notation.
