Javascript vs HTML and CSS:

HTML and CSS are static forms of code, this means they dont have animated sections or lack interaction between users and the site. Java script on the other hand allows interactive and dynamic interactions and features within websites. Javascript allows loops to be created, data to be sotred and accessed, and compute algebra. While Javascript has many benefits, it can feel like HTML and CSS's uptight cousin who likes to throw in the towel if your diction or syntax are even slightly off.
To summarise:

  • HTML gives a structure
  • CSS allows for style and design
  • Javascript allows programmers to create unique and interactive experiences


    • Control Flow and Loops:

    Control Flow is an important concept to be mindful of as it is the order in which the computer proceeds to execute statements. Example: Imagine before you went to bed each night, you wrote a list of things to do the next day after you wake up. This list included activities like brushing your teeth, getting dressed, having breakfast and heading to work. Now if you were to write these in a radom order you may get dressed once you have left the house and are at work in fornt of your work colleagues. However if you were to prioritise these activities then you could ensure you don't rock up to work without breakfast and naked.
    Control Flow can be thought of in this manner. It causes priorities for computers to process the instructions.

    Loops are a type of code that will repeat until interrrupted or told to stop. Loops reduce the amount of coding required to do repeatable functions. Loops allow a more refined and compact level of code to repeat coding blocks.
