Javascript vs HTML and CSS:

HTML and CSS are static forms of code, this means they dont have animated sections or lack interaction between users and the site. Java script on the other hand allows interactive and dynamic interactions and features within websites. Javascript allows loops to be created, data to be sotred and accessed, and compute algebra. While Javascript has many benefits, it can feels like HTML and CSS's uptight cousin who likes to throw in the towel if your diction or syntax are even slightly off.
To summarise:

  • HTML gives a structure
  • CSS allows for style and design
  • Javascript allows programmers to create unique and interactive experiences
